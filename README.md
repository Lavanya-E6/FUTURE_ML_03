# Resume Screening System using Machine Learning

This project builds a Resume Screening and Candidate Ranking System using Machine Learning and NLP techniques.

The system analyzes resume text, compares it with a job description, calculates similarity scores, ranks candidates, and identifies missing skills.

## Technologies Used

* Python
* Pandas
* Scikit-learn
* TF-IDF Vectorization
* Cosine Similarity
* Matplotlib

## Features

* Resume text preprocessing
* Job description matching
* Candidate ranking
* Skill gap analysis
* Visualization of match scores

## Output

The system provides:

* Candidate ranking based on role fit
* Match percentage
* Missing skills analysis
